Week Four Notes

All of the fields within a form are created inside the form tag, including the end button. Forms are secondary in referencing if they have the same name as a method. You can use changed, blur, or focus as events that can occur as the user interacts with a form. You can access any parts of a form in JS using index notation. For form validation, JS should only be used for usability. Server-side validation should be used for validation before being saved to a database. You can make fields required using HTML but it is not supported by all browsers so be aware of this. There are workarounds. You can also disable the submit button with a function is a field is not entered correctly.
